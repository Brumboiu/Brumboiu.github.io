GitHub este un serviciu de gazduire web pentru proiecte de dezvoltare a software-ului care utilizeaza sistemul de control al versiunilor Git. GitHub ofera planuri tarifare pentru depozite private, si conturi gratuite pentru proiecte open source. Site-ul a fost lansat in 2008 de catre Tom Preston-Werner, Chris Wanstrath, si PJ Hyett.

E-learning

E-learning (sau eLearning) reprezinta utilizarea de media electronica si tehnologii informationale si de comunicatie (ICT) in educatie. E-learning la general include toate formele de tehnologie educationala in educatie si invatare. E-learning este sinonim cu multimedia learning, technology-enhanced learning (TEL), computer-based instruction (CBI), computer-based training (CBT), computer-assisted instruction or computer-aided instruction (CAI), internet-based training (IBT), web-based training (WBT), online education, virtual education, virtual learning environments (VLE) (which are also called learning platforms), m-learning, and digital educational collaboration. These alternative names emphasize a particular aspect, component or delivery method.
